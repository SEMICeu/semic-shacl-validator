# semic-shacl-validator

The online service is accessible as follows:
* Web UI: https://www.itb.ec.europa.eu/shacl/semic-shacl/upload.
* REST API (Swagger docs): https://www.itb.ec.europa.eu/shacl/swagger-ui.html (use 'semic-shacl' as the domain value in requests).
* REST API (Hydra docs): https://www.itb.ec.europa.eu/shacl/semic-shacl/api.
* SOAP API: https://www.itb.ec.europa.eu/shacl/soap/semic-shacl/validation?wsdl.
